## Shell redirections Scripts

I/O Redirection (input / output) commands help us redirect a result to other locations like, files, devices, and even to input of other commands

0. 0-hello_word

	Prints "Hello, World" to the display

1. 1-confused_smiley

2. 2-hellofile

	Displays the content of /etc/passwd file

3. 3-twofiles

	Displays the content of file /etc/passwd and /etc/hosts

4. 4-lastlines

	Displays the last 10 lines of /etc/passwd

5. 5-firstlines

	Displays the first 10 lines of /etc/passwd
	
6. 6-third_line

	Displays the third line of file 'iacta'

7. 7-file

	Creates a file named exactly '\*\\'"Best School"\'\\*$\?\*\*\*\*\*:)' containing the text "Best School"

9. 9-duplicate_last_line

	Duplicates the last line of the file 'iacta'

10. 10-no_more_js

	Deletes all the regular files (not directories) with .js extension, that is present in the directory and all its subfolders.

11. 11-directories

	Counts the number of directories and sub-direcotries in the current direcotry

12. 12-newest_files

	Displays the 10 newest files in the current direcotry

13. 13-unique

	Takes a list of words and only prints the words that appear once

14. 14-findthatword

	Displays the lines containg the pattern 'root' from the file /etc/passwd

15. 15-countthatword

	Displays the number of lines that contain the pattern 'bin' in file /etc/passwd

16. 16-whatsnext

	Displays the pattern 'root' and 3 lines after in the file /etc/passwd

17. 17-hidethisword

	Displays all the lines in the file /etc/passwd that do not contain the pattern 'bin'

18. 18-letteronly

	Displays all lines in the file /etc/ssh/sshd_config starting with a letter

19. 19-AZ

	Replace all characters A and c from input to Z and e respectively

20. 20-hiago

	Moves all the letters c to C from input

21. 21-reverse

	Reverse its input

22. 22-users_and_homes

	Displays all users and their home directories, sorted by users from the file /etc/passwd

23. 100-empty_casks

	Finds all empty files and directories  in the current direcotry and all sub-directories

24. 101-gifs

	List all files with a .gif extension in the current directory and all its sub-directories.
